Offrez dès à présent aux clients la possibilité de payer avec TWINT dans votre magasin de ferme, sur votre champ de fleurs ou à votre étal de marché.
Marche à suivre
Placez un ou plusieurs autocollants de code QR TWINT résistants aux intempéries dans votre point de vente.
Les clients scannent le code QR, saisissent le montant et confirment le paiement par TWINT.
Le montant est crédité sur votre compte. Vous recevez une confirmation par SMS ou par e-mail.
Vos avantages
Pour chaque transaction traitée via le système de paiement TWINT, un pourcentage fixe du volume de transactions de 1,3 % s’applique pour le partenaire contractuel.
Le panneau de ferme, le support, l’oriflamme publicitaire, ainsi que les autocollants de code QR de TWINT vous sont fournis gratuitement.
Idéal également pour les points de vente en libre-service.
Plus de 3 millions d’utilisateurs et utilisatrices au pouvoir d’achat élevé.
Aucun vol de caisse possible : les paiements sont comptabilisés directement et en toute sécurité sur votre compte.
