Air Navigation Pro vous laisse utiliser toutes ses options avec MS Flight	Simulator X (FSX).
Contents
1 Localisation des défaillances
Pour ce faire, vous devez suivre les étapes suivantes:	
Vous assurez d’installer le paquet “Acceleration” avec FSX (ou le SP1 et le SP2, que vous pouvez obtenir sur Internet).
Télécharger et installer le plugin à	partir de notre site Web sous la section «Support»: http://airnavigation.aero/support/
Si vous avez des problèmes	lors de l’installation du plugin, exécutez le logiciel comme “Administrateur” (en faisant un clic droit sur l'icône du fichier).
Télécharger et installer l’application “Bonjour”: http://support.apple.com/kb/DL999
Assurez-vous d’avoir votre PC et votre iPad/iPhone connectés au MÊME réseau wifi.
Démarrer le logiciel FSX et accepter tous les messages qui apparaîtront par rapport à l’installation du plugin.
Démarrer Air Navigation Pro sur votre iPad/iPhone. Assurez-vous de n’avoir que le capteur “Flight Simulator X” activé dans le menu «Tools» (il faut en désactiver les autres y compris celui appelé “iOS Location Services”).
Commencer un vol sur FSX et presser la touche “ALT”, puis sélectionner l‘option Start Air Navigation bridge depuis le menu “Add-ons”.
Note: assurez-vous de n’avoir aucun Firewall qui puisse bloquer la connexion.
Localisation des défaillances
1.	Contrôlez que le fichier "AirnaveBridgeFSX.dll" soit dans le dossier \AirnvBridgeFSX au sein de la structure P3D / FSX directory.
Vérifiez que les installateurs succédé modifie le DLL.XML ( localisé dans c:\Utilisateurs\<Votre Nom>\Appdata\Roaming\Lockheed Martin\Prepar3D v2\). Dans tous les cas, éditez le DLL.XML pour vérifier que cette section existe:
<Launch.Addon> <Name>Air Nav Bridge FSX</Name> <Disabled>False</Disabled> <Path>AirnavBridgeFSX\AirnavBridgeFSX.dll</Path> </Launch.Addon>
Si ce n’est pas le cas, corriger/éditer DLL.XML pour refléter.
Note: Il peut aussi s’agir d’une simple erreur, comme une faute dans la ligne du fichier DLL. Exemple: A la place d’avoir "Air Nav", vous lisez "Air Mav".
Si vous rencontrez encore des problèmes:
Si vous rencontrez une erreur dans le nom du fichier donné dans le message d’erreur FSX, par exemple « AirmavBridgeFSX… », avec un « m » à la place du « n », vous devez suivre la procédure suivante: Renommer TOUS les fichiers présents dans le plugin FSX Air Nav Pro comportant la même erreur (remplacer le « m » par un « n »). Par exemple: si vous avez AirnavBridgeFSX.dll, renommer le fichier par AirmavBridgeFSX.dll
Si vous continuez à rencontrer des problèmes:
Essayer de retirer cette clé du registre:
[HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\AppCompatFlags\Layers] "C:\\Program Files (x86)\\Microsoft Games\\Microsoft Flight Simulator X\\fsx.exe"="$ IgnoreFreeLibrary<AirnavbridgeFSX.dll> HIGHDPIAWARE"
