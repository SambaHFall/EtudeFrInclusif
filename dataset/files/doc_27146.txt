Commande d'un service VPS Windows non géré
Nos VPS Windows Cloud non gérés vous donnent un serveur Windows avec un environnement de bureau.Vous obtenez un accès complet au serveur, ainsi que des ressources dédiées indépendamment des autres utilisateurs.Ce serveur est entièrement à vous à utiliser comme vous le souhaitez.Ce serveur n'inclut toutefois aucun support de Hostwinds.
Commander un plan VPS Windows non géré
Pour passer une commande pour un VPS Linux non gérés, vous pouvez commencer par l'hostwinds ' Site Internet ou votre hostiewinds Espace Client Mais les deux vous mèneront à la même page de commande.
Dans cet article cependant, nous continuerons avec les étapes Un client existant doit prendre pour commander un nouveau plan d'hébergement VPS Linux non exploité.
Remarque: Pour ajouter un plan d'hébergement VPS Linux non géré à votre compte, vous devez vous connecter à l'hostwinds. Espace Client.Dans la page d'accueil de la zone client, cliquez sur le lien de dépose des services dans le menu supérieur du tableau de bord.Sélection de la commande Le lien de nouveaux services dans la liste déroulante vous déplace vers la page d'achat de service, où vous êtes présenté avec tous les services d'Hostwinds disponibles à l'achat.
En haut à gauche de la page, la liste déroulante Catégories montre une liste de liens.En cliquant sur le lien SSD Cloud Server pour commencer le processus de l'achat.Les options VPS gérées que nous fournissons apparaîtront à droite.
Informations client
Vous serez invité à entrer dans votre nom, votre entreprise, votre numéro de téléphone et votre adresse.
Géré / non géré
Hostwinds vous fournit la possibilité de comparer les prix entre les options gérées et non gérées en cliquant sur l'un des boutons.
Cycle de facturation
Vous devrez sélectionner le Cycle de facturation suivant.Cela déterminera les périodes de temps entre chaque facture que vous recevez pour votre plan.Hostwinds offre plusieurs longueurs de cycles de facturation décrites ci-dessous:
Mensuel
Trimestriel
Semestriellement
Biennalement
Triennalement
Emplacement
Vous devez sélectionner l'emplacement de votre hébergement ensuite.Plus l'emplacement est proche de votre marché cible ou de votre source principale d'utilisateurs, plus vite votre vitesse.Par exemple, si vos utilisateurs sont principalement européens, nous vous recommandons d'utiliser notre emplacement d'Amsterdam pour le meilleur.Vous pouvez tester les temps de latence et de téléchargement de votre emplacement sur notre Centres de données page.
Système opérateur
De là Hostwinds vous fournit un choix de chaque système d'exploitation que nous prenons en charge l'installation sur notre plate-forme d'hébergement.
Adresses IP
Choisissez parmi 1 à 250 adresses IP pour votre serveur dans la liste déroulante.
Adresses IP protégées DDOS
Hostwinds propose une protection DDOS à titre d'adresses IP distinctes pouvant filtrer des demandes illégitimes.Nous offrons 5 initialement dans la liste déroulante
Adresses IP C-Class
Choisissez parmi 1 à 250 adresses IP de classe C pour votre serveur dans la liste déroulante.
Paquet
La section Package fournit une comparaison de tous les packages liés à l'hébergement partagé dans une liste déroulante.Notez que la modification du cycle de facturation Dropdown vous indiquera le prix du colis que vous choisissez.
Addons
Surveillance des hostwinds: notre Système de surveillance Configurez des contrôles en temps réel pour votre service d'hébergement.Si ces chèques prennent des échecs à tout moment, ils ouvrent automatiquement un billet à l'un de nos agents de première ligne, qui commencent immédiatement à travailler sur la solution.
Sauvegardes de nuages d'hébergement Web: Notre Service de sauvegarde Fournit une sauvegarde nocturne de vos services stockés dans le stockage de l'objet Cloud Hostwinds.Celles-ci sont utilisées pour restaurer votre site si quelque chose se passe lorsque vous y travaillez.
Remarque: les sauvegardes sont sélectionnées pour votre plan par défaut et vous pouvez les désactiver à l'achat en désélectionnant l'addon de surveillance.Nous recommandons que des sauvegardes soient faites pour tout service de sauvegarde contre le risque potentiel de perte de données.
Licences logicielles
Hostwinds propose des licences CPanel et des licences sadculaires pour l'achat mensuel.
CPanel: CPanel est la norme pour les GIIS du logiciel du panneau de configuration.
Softaculeux: auto-installateur satinaculeux fournit une installation en un clic pour des centaines d'applications Web.
Informations de paiement
Enfin, entrez vos informations de paiement dans la section suivante.Vérifiez que vous avez sélectionné le bon package.Si vous avez sélectionné des addons ou sélectionné plusieurs domaines à acheter, ceux-ci s'affichent dans la panne de prix.Veuillez lire et accepter d'accueillir les conditions d'utilisation du contrat de service et cliquez sur le bouton de commande complet bleu en bas de la page.
