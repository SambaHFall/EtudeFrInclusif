Inscrivez votre pseudo. Choisissez la table avec laquelle vous souhaitez jouer ou réviser et répondez aux questions le plus rapidement possible pour gagner votre médaille (de bronze, d’argent et d’or) ! Une fois le jeu fini, vous pouvez enregister votre score pour le comparer aux autres joueurs. dans le classement du jeu des tables de multiplication.
C’est une manière ludique d’apprendre et de réviser les tables de multiplication. L’apprentissage des tables est un travail de longue halaine qui necessite un long travail de fond, alors pourquoi ne pas utiliser le jeu pour le rendre moins rébarbatif.
