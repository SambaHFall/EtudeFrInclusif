Pourquoi utiliser un logiciel de gestion documentaire (GED) ?
Un logiciel de gestion documentaire (GED) permet la gestion éléctronique des documents : fichiers textes, tableurs Excel, sauvegarde, partage, conformité. Les logiciels de gestion documentaire permettent de gérer tous les documents d’une entreprise, de les stocker, de les centraliser, de les partager, de les consulter, de les modifier. (GED : Gestion Electronique de Documents / EDM : Electronic Document Management ). Les principaux logiciels métiers sont : Google Drive (part de marché : environ 28,40%), IBM FileNet (9,40%), DocuWare (6,20%), Laserfiche (3,80%), Atlassian Confluence (2,70%), EMC Captiva (2,40%), Alfresco (1,80%), Worldox (1,20%), EVault (0,70%), KnowledgeLake (0,50%), Dropbox Paper (0,10%) …
Nous avons identifié près de 333 logiciels dans cette catégorie : le tarif moyen est de 18,23 euros par licence utilisateur.
