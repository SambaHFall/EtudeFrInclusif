SQL Server 2008 Express Edition est une version gratuite, facile à utiliser, légère et intégrable de SQL Server 2008. Gratuit à télécharger, gratuit à redistribuer, gratuit à intégrer et facile à utiliser immédiatement pour les nouveaux développeurs.
SQL Server 2008 Express Edition présente les avantages suivants :
Rapide à télécharger et à installer : Il s'agit d'un téléchargement compact, et vous pouvez être opérationnel en 20 minutes seulement.
Une administration simplifiée : Réglez automatiquement votre base de données pour obtenir des performances optimales. Pour plus de commodité, vous pouvez utiliser Microsoft Update pour recevoir automatiquement les derniers correctifs et mises à jour, ce qui vous permet de bénéficier d'un niveau de sécurité élevé et des dernières fonctionnalités.
Outil de gestion gratuit : Studio de gestion du serveur SQL : Vous pouvez télécharger le SQL Server Management Studio Express, un outil de gestion simple et efficace pour SQL Server Express. Avec SQL Server Management Studio, vous bénéficiez d'une expérience de gestion cohérente sur toutes les éditions de SQL Server 2008.
Intégration transparente avec les éditions Visual Studio Express.
Facile à déployer : Déplacez ou copiez facilement vos bases de données SQL Server Express vers un autre ordinateur ou un serveur hébergé grâce au déploiement XCopy et ClickOnce.
