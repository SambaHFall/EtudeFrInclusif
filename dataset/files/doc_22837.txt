Verres Blanc Cat.1
Ce coloris est utilisé pour se protéger des projections physiques telles que, gravillons, moustiques, poussières, etc. Il n’a aucune incidence sur l’ensoleillement.
Orange Cat.1
Ce coloris augmente les contrastes et minimise la fatigue visuelle. IL est conseillé par temps gris, brouillard, mauvaise visibilité.
Jaune Cat.1
Ce coloris absorbe la lumière bleue du spectre lumineux. Il donne une perception de profondeur ainsi qu’un excellent contraste lors de faible luminosité. Il est idéal pour la conduite par temps gris, brouillard, mauvaise visibilité.
Vert Cat.3
Ce coloris offre une courbe de transmission qui ressemble beaucoup à celle de l’œil humain. Il a une bonne absorption, tant dans les rayons infrarouges que dans les rayons ultra-violets. Il procure une bonne vision des contrastes, cette teinte diminue la fatigue visuelle lorsque la lumière est trop intense. Ce coloris convient parfaitement aux porteurs de lentilles de contact.
Fumé Cat.4
Ce coloris plus foncé que le Fumé Cat.3 sera utilisé pour les conditions d’ensoleillement extrême. Il est absolument déconseillé pour la conduite. Il est utilisé pour le ski, le bateau, etc. Idéal pour les Hypermétropes.
Fumé Cat.3
Ce coloris réduit l’intensité lumineuse sans dénaturer les couleurs, car cette teinte transmet uniformément la lumière à travers tout le spectre lumineux. Il est idéal pour les Hypermétropes.
Brun Cat.3
Ce coloris améliore les contrastes et réduit l’éblouissement pour les conditions d’ensoleillement. Il offre une excellente protection solaire dans la majorité des cas, par ailleurs, le brun absorbe les longueurs d’onde dans le bleu. Ce coloris est particulièrement adapté pour les Myopes.
Brun Cat.4
Ce coloris plus foncé que le brun Cat.3 sera utilisé pour les conditions d’ensoleillement extrême. Il est absolument déconseillé pour la conduite. Il est utilisé pour le ski, le bateau, etc. Idéal pour les Myopes.

Quelle taille commander ?

Mesurer la distance entre les branches.
