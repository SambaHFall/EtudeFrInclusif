Comment Fonctionne Votre Cache Pleine Page ?
2 novembre 2021
Le cache est géré côté serveur par nos soins (mise en cache pleine page). Il n’y a pas besoin de plugins de cache typiques. Le cache pleine page est réglé à 1 heure par défaut, mais nous pouvons l’affiner pour vous.
Cache WordPress de Kinsta
Le MU plugin Kinsta est installé par défaut sur tous les sites hébergés Kinsta. Ce n’est pas un plugin de mise en cache typique, car il fonctionne au niveau du serveur. Il vous permet de purger le cache de l’objet, le cache pleine page ou le cache complet, directement depuis la barre d’administration. Vous pouvez également contrôler les paramètres de purge automatique lorsque les messages et les commentaires sont publiés ou mis à jour.
Pour plus de détails, veuillez consulter notre article détaillé sur le sujet : Kinsta gère la mise en cache WordPress pour que vous n’ayez pas besoin de le faire
Le plugin vous montre également les hits et les manques de mémoire cache, la quantité de mémoire actuellement utilisée et la mémoire totale disponible.
