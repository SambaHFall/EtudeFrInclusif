Le Blender Institute dévoile enfin son nouveau court-métrage libre, Cosmos Laundromat – First Cycle, également connu sous le nom de code Projet Gooseberry.
Réalisé par Mathieu Auvray, le court met en scène Franck, un mouton suicidaire, sur une île désolée. Il rencontrera Victor, un homme qui lui fera une offre étonnante…
Le projet est à voir comme le pilote d’un possible long-métrage ; le projet devait d’ailleurs initialement directement déboucher sur un long-métrage animé. Faute de moyens suffisants, seul un court a pu être produit.
Vous pourrez découvrir sur le site du projet de nombreux articles sur les coulisses de la production, ainsi que la liste complète des artistes impliqués.
